# Organization Knowledge Assistant

A Retrieval-Augmented Generation (RAG) based question-answering system built using Python, Sentence Transformers, FAISS, and Pandas.

## Features

* Natural language question answering
* Semantic search using embeddings
* FAISS vector similarity retrieval
* Typo-tolerant company search
* Organization information lookup
* Foundation for Gemini-powered responses

## Technologies Used

* Python
* Pandas
* Sentence Transformers
* FAISS
* NLP
* RAG

## Example Questions

* Where is Company_20 located?
* What is the revenue of Company_20?
* Tell me about Company_15.
